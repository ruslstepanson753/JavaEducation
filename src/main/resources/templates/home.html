О приложении

Это приложение написано с целью, чтобы начинающие java девелоперы могли готовиться к собеседованиям. В приложении собрано 768 вопросов, разбитых по темам.
- предпочитаемую тему можно настроить
- пользователь сам выбирает, на сколько хорошо он владеет вопросом
- помеченные "знаю хорошо" вопросы удаляются из базы пользователя и далее не отображаются
